Key insights

  • Fri 4/17 beats Thu 4/16 for the outbound. Friday cheapest = $179 (Delta LGA evening). Thursday cheapest = $194 (Delta JFK morning).
  • Sunday returns are the expensive leg of the trip. MSY → NYC nonstops start at $465; MSY → LAX nonstops start at $344. Both Sundays are flagged "currently high" — unlikely to drop.
  • Only 5 nonstops exist MSY → LAX on Sunday. Mom's return dictates schedule flexibility, not yours.
  • Delta is the glue airline: one carrier covers outbound (Delta LGA 4:40 PM), Natalia's return (Delta JFK 7:55 PM), and Mom's return (Delta LAX 7:59 PM). Worth booking on delta.com for easier changes and SkyMiles.
  • Evening departures sync up Sunday: Natalia 7:55 PM + Mom 7:59 PM from MSY = one Uber to the airport, one emotional goodbye, done.
  • LGA is the best NYC airport for Brooklyn. ~30 min by Uber from Downing St. Avoids EWR (NJ taxes, farther) and JFK (farther east).
